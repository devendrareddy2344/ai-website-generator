# GenSite AI

GenSite AI is an AI-powered website generator that converts natural language
prompts into fully functional responsive websites using Gemini LLM.

## Features
- Chat-based input
- Gemini-powered code generation
- Live preview
- HTML/CSS/JS export

## Tech Stack
Frontend: HTML, CSS, JS  
Backend: FastAPI  
AI Model: Gemini Pro  

## Architecture
User → Frontend → FastAPI → Gemini API → Generated Website

## Deployment
Frontend: Vercel  
Backend: Render
